## 패치 노트
### ny-a.1.0 | 2026.03.06

**전체 리메이크**
- 파이썬(Streamlit) 기반 웹사이트를 C#(WinUI 3) 기반 데스크탑 앱으로 전면 재작성
- 기술 스택 확정: C# / WinUI 3 / MVVM / Supabase .NET SDK

**집행부 기능 이식**
- 로그인 화면 (비밀번호 인증)
- 승인 대기 탭: 대기 중 신청 목록 확인, 담당자 지정, 승인/반려 처리
- 진행 중 대여 탭: D-Day 표시, 반납 완료 처리, 대기 복원
- 전체 이력 탭: 전체 대여 기록 DataGrid
- 장비 목록 탭: 인벤토리 편집
- 설정 탭: 관리자 비밀번호 변경

**신규 기능**
- Windows 토스트 알림: Supabase Realtime 기반, 새 대여 신청 접수 시 즉시 알림 발송

**인프라**
- Supabase 테이블 구조 유지 (Rentals, Inventory, Settings)
- appsettings.json을 통한 자격증명 관리
- .gitignore로 자격증명 파일 보호

### ny-a.1.2 (2026.03.06)

**레이아웃 개선**

- 대여 신청 양식을 단일 세로 스크롤 구조에서 2~3열 Grid 기반 레이아웃으로 변경하여 가로 공간 활용도를 높이고 스크롤 없이 한 화면에서 모든 항목 입력 가능하도록 개선

**기능 변경**

- 앱 시작 시 비밀번호 입력 없이 바로 메인 화면(캘린더/대여 신청)에 진입하도록 변경, 비밀번호는 '집행부 관리' 탭 진입 시에만 ContentDialog 팝업으로 요구
- 캘린더 컬러 라인 표시 순서를 대여 기간(일수) 내림차순으로 정렬 — 더 오래 빌리는 대여 건이 항상 상단에 표시
- 카메라 바디 없이 렌즈만 선택하여 대여 신청 가능하도록 변경 — 바디 미선택 시 전체 렌즈 목록 표시, 바디 선택 시 호환 렌즈 필터링

**UI 수정**

- 대면 가능 시간 라벨 정리: '대여 시작' → '대여', '반납 시작' → '반납'으로 변경, 종료 시간 ComboBox의 Header 제거
- DatePicker 월 표시를 영어(March 등)에서 한국어(3월 등)로 변경
- 대면 가능 시간 ComboBox 수평 정렬 불일치 버그 수정

**버그 수정**

- 집행부 관리 탭에서 승인 대기 항목 선택 시 '선택된 신청이 없습니다'로 표시되던 버그 수정 — `OnSelectedRentalChanging`에서 `OnSelectedRentalChanged`로 프로퍼티 변경 알림 이동

### ny-a.1.3 (2026.03.11)
**테마 시스템 구축**
- `Themes/` 폴더에 BrandColors, ControlStyles, CardStyles ResourceDictionary 생성 — 기존 웹사이트(style.css)의 브랜드 컬러 팔레트를 WinUI 3로 이식
- Light/Dark 모드 브랜드 컬러 정의: Light `#B2DFDB` 민트 / Dark `#004D40` 딥 틸 계열
- Button, ComboBox, TextBox, ListView, TabView 등 주요 컨트롤에 커스텀 스타일 적용 (CornerRadius 8~12, 브랜드 컬러 강조)
- 커스텀 타이틀바 적용 (`ExtendsContentIntoTitleBar`) — 브랜드 컬러 배경 + 앱 타이틀 표시
- D-Day 배지 커스텀 컨트롤(`DDayBadge`) 생성 — 여유(초록)/주의(주황)/긴급(빨강) 3단계 자동 컬러 적용
- 전 페이지에 카드 스타일, 섹션 헤더 스타일, 배지 컬러 일괄 적용
**레이아웃 개선**
- 대여 신청 양식을 2컬럼 레이아웃으로 재설계 — 좌측: 장비 선택/액세서리/추가 요청, 우측: 신청자 정보/대여 기간/대면 시간 배치로 세로 스크롤 최소화
**UI 수정**
- 승인 대기 탭 이름 옆에 빨간 원형 배지로 대기 건수 표시
- 승인 대기 목록 카드형 레이아웃 적용 — 신청자/연락처 상단, 장비명 중단, 기간 배지(초록)+액세서리 배지(회색) 하단 구조
- 승인 대기 카드 전체 크기(패딩, 텍스트) 약 20% 확대하여 가시성 향상
- 승인 대기 카드 배경색을 앱 배경보다 밝게 조정하여 카드 구분 명확화
- 신청 처리 패널의 InfoBar를 Border 카드로 교체하여 장비 정보 표시 정리
- 로그인 카드, DataGrid 헤더, 설정 페이지 버튼 등에 브랜드 스타일 적용
**버그 수정**
- 승인 대기 카드 선택 시 하이라이트 영역이 카드 Border와 불일치하던 버그 수정
- 승인 대기 카드 가로 길이가 내용에 따라 달라지던 버그 수정 — 전체 너비로 통일
- 라이트 모드 전환 시 다크 모드 초록색(#004D40 등)이 민트색으로 바뀌지 않던 버그 수정 — 하드코딩 색상을 ThemeResource로 전환
- 대여 신청 탭에서 렌즈 선택 시 좌측 '장비 선택' 영역이 좁아지던 레이아웃 버그 수정
**기능 추가**
- 대여 신청 탭에서 카메라 바디 및 렌즈 선택을 취소(초기화)할 수 있는 ✕ 버튼 추가
